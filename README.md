# Superstore Sales Dashboard (Power BI)

Fully interactive sales analytics dashboard built from raw data.

## Features
- 4 KPI cards (Total Sales, Profit, Margin %, Orders)
- Sales & Profit trend over years
- Top products by sales
- Interactive slicers (Category + Year)
- Built using Power Query, DAX measures, and modern layout

## Tools Used
- Power BI Desktop
- Power Query (data cleaning)
- DAX (custom measures)

## How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Click slicers to filter instantly!

Open to Junior Data Analyst / BI roles
